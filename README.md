[**Course material in Plussa**](https://plus.tuni.fi/comp.ce.420/spring-2025/)

New material is added by the staff periodically to the [template repository](https://course-gitlab.tuni.fi/compce420-spring2025/group_template_project)\
To merge the changes to your repo:
```
git remote add course_upstream https://course-gitlab.tuni.fi/compce420-spring2025/group_template_project.git
git fetch course_upstream
git merge course_upstream/main
```
